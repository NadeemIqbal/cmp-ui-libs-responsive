AdaptiveNavigationItem

class AdaptiveNavigationItem(val label: String, val icon: @Composable () -> Unit, val selectedIcon: @Composable () -> Unit = icon)(source)

A single destination in AdaptiveNavigation.

Icons are @Composable () -> Unit slots so consumers can render any icon source they prefer (Material Icons, custom drawables, Compose Resources) without this library taking a dependency on a specific icon set.
